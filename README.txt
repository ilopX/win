Соединение платформы Windows с прекрасным языком Дарт. Может использоваться внутри приложения флаттер или без него. 

Особенности:
Не кроссплатформенный.
Без с++.

Задачи:
Покрыть спектр низкоуровневых windows апи обернув их в высокоуровневые обертки. 

План:
Врапперы
Window
App
RegEdit
Print
Dialogs
Power
FlutterEngine
DartEngine
Media Fundation

Утилиты
win:create [flutter | service | app] - создает скелет приложения
win:run - запуск флаттер приложения
win:build [ --single] - сборка флаттер приложения или упаковка в инсталлятор
